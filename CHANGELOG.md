## [0.1.0] - Initial Release

### Added
- Role-based access control (System Admin, Manager, Employee)
- Schedule management with shift creation and editing
- Employee management and invitation system
- Shift template creation and management
- Employee preference submission and review
- Shift swap request system
- Real-time updates via Supabase
- Responsive design with dark mode support
- User profile and company settings
- Admin dashboard for platform management
- Error boundaries for graceful error handling
- Test coverage for critical components
- Pagination for large data sets

### Tech Stack
- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- shadcn/ui component library
- Supabase for backend and authentication
- TanStack Query for data fetching
- React Hook Form with Zod validation
- Vitest for testing
